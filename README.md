# Codex에서 사용하는 방법

1. 이 압축파일 안의 `AGENTS.md`와 `CURRENT_TASK.md`를 작업 중인 프로젝트의 최상위 폴더에 넣습니다.

예:

```text
jkcf-renewal/
├─ AGENTS.md
├─ CURRENT_TASK.md
├─ index.html
├─ about.html
├─ activities.html
├─ stories.html
├─ news.html
├─ membership.html
├─ css/
├─ js/
└─ images/
```

2. Codex에서 `jkcf-renewal` 폴더를 프로젝트로 엽니다.

3. Codex에 아래 한 문장만 입력합니다.

```text
AGENTS.md와 CURRENT_TASK.md를 읽고 현재 작업을 진행해줘.
```

4. 다음 단계로 넘어갈 때는 `CURRENT_TASK.md`의 내용만 새 작업 내용으로 교체하면 됩니다.

주의:
- `images/logo_header_pc.png`는 실제 프로젝트의 `images` 폴더 안에 직접 넣어야 합니다.
- 현재 `CURRENT_TASK.md`는 PC 헤더와 내비게이션만 작업하도록 설정돼 있습니다.
- 모바일 메뉴, 비주얼, 섹션, 푸터는 아직 만들지 않도록 제한돼 있습니다.
